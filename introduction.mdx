    <h2 className="text-3xl sm:text-4xl font-bold text-gray-900 dark:text-white mb-4">
      Quick Start
    </h2>
    <p className="text-lg text-gray-600 dark:text-gray-400 max-w-2xl mx-auto">
      Get up and running with Based Roop in minutes
    </p>
  </div>

  <Steps>
    <Step title="Install Dependencies">
      Based Roop requires Python 3.9+ and FFmpeg. Install the required packages:
      
      ```bash
      pip install -r requirements.txt
      ```
      
      <Note>
        On first run, Based Roop will automatically download AI models (~300MB). This is a one-time setup.
      </Note>
    </Step>

    <Step title="Prepare Your Media">
      You'll need two files:
      - **Source image**: A photo containing the face you want to use
      - **Target media**: An image or video where you want to replace faces
      
      Make sure your source image clearly shows a single face for best results.
    </Step>

    <Step title="Run Face Swapping">
      Launch the GUI interface to get started:
      
      ```bash
      python run.py
      ```
      
      Or use the CLI for automation:
      
      ```bash
      python run.py --source face.jpg --target video.mp4 --output result.mp4
      ```
      
      <Accordion title="Expected Output">
        The processing will show real-time progress:
        
        ```
        [ROOP.CORE] Extracting frames...
        [ROOP.FACE-SWAPPER] Progressing...
        [ROOP.CORE] Creating video with 30.0 fps...
        [ROOP.CORE] Processing to video succeed!
        ```
      </Accordion>
    </Step>

    <Step title="Enhance Results (Optional)">
      Add face enhancement for higher quality output:
      
      ```bash
      python run.py --source face.jpg --target video.mp4 --output result.mp4 \
        --frame-processor face_swapper face_enhancer
      ```
      
      This applies GFPGAN enhancement to improve facial details and quality.
    </Step>
  </Steps>
</div>
