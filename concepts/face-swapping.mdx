## Overview

Based Roop uses the InsightFace framework to detect faces and swap them in images and videos. The face swapping processor is implemented in `face_swapper.py` and uses the `inswapper_128.onnx` model for high-quality face replacement.

## InsightFace Model

The face swapper loads the InsightFace model on demand using a thread-safe singleton pattern:

```python
def get_face_swapper() -> Any:
    global FACE_SWAPPER
    
    with THREAD_LOCK:
        if FACE_SWAPPER is None:
            model_path = resolve_relative_path('../models/inswapper_128.onnx')
            FACE_SWAPPER = insightface.model_zoo.get_model(
                model_path, 
                providers=roop.globals.execution_providers
            )
    return FACE_SWAPPER
```

The model is automatically downloaded from HuggingFace during the `pre_check()` phase:

```python
def pre_check() -> bool:
    download_directory_path = resolve_relative_path('../models')
    conditional_download(download_directory_path, [
        'https://huggingface.co/henryruhs/roop/resolve/main/inswapper_128.onnx'
    ])
    return True
```

## Face Detection

Based Roop provides two methods for face detection, both implemented in `face_analyser.py`:

### Single Face Detection

`get_one_face()` returns the leftmost face in a frame:

```python
def get_one_face(frame: Frame) -> Any:
    face = get_face_analyser().get(frame)
    try:
        return min(face, key=lambda x: x.bbox[0])
    except ValueError:
        return None
```

This method:
- Uses the `buffalo_l` model from InsightFace
- Detects at 640x640 resolution
- Returns the face with the smallest x-coordinate (leftmost)
- Returns `None` if no faces are found

### Multiple Face Detection

`get_many_faces()` returns all detected faces:

```python
def get_many_faces(frame: Frame) -> Any:
    try:
        return get_face_analyser().get(frame)
    except IndexError:
        return None
```

This method returns all faces detected in the frame, enabling batch processing.

## Face Swapping Implementation

The core swapping logic is implemented in the `swap_face()` function:

```python
def swap_face(source_face: Face, target_face: Face, temp_frame: Frame) -> Frame:
    return get_face_swapper().get(
        temp_frame, 
        target_face, 
        source_face, 
        paste_back=True
    )
```

**Parameters:**
- `source_face`: The face to copy from (from your source image)
- `target_face`: The face to replace in the target frame
- `temp_frame`: The frame being processed
- `paste_back=True`: Seamlessly blends the swapped face back into the frame

## Single vs. Many Faces Mode

The `process_frame()` function handles both single and multiple face swapping:

```python
def process_frame(source_face: Face, temp_frame: Frame) -> Frame:
    if roop.globals.many_faces:
        if many_faces := get_many_faces(temp_frame):
            for target_face in many_faces:
                temp_frame = swap_face(source_face, target_face, temp_frame)
    elif target_face := get_one_face(temp_frame):
        temp_frame = swap_face(source_face, target_face, temp_frame)
    return temp_frame
```

## Processing Pipeline

<Steps>
  <Step title="Load Source Face">
    Extract the face from the source image using `get_one_face()`
    ```python
    source_face = get_one_face(cv2.imread(source_path))
    ```
  </Step>
  
  <Step title="Detect Target Faces">
    Find faces in the target frame using either `get_one_face()` or `get_many_faces()`
  </Step>
  
  <Step title="Swap Faces">
    Replace each target face with the source face using the InsightFace model
  </Step>
  
  <Step title="Write Output">
    Save the processed frame back to disk
    ```python
    cv2.imwrite(temp_frame_path, result)
    ```
  </Step>
</Steps>

## Validation

The face swapper performs validation before processing:

```python
def pre_start() -> bool:
    if not is_image(roop.globals.source_path):
        update_status('Select an image for source path.', NAME)
        return False
    elif not get_one_face(cv2.imread(roop.globals.source_path)):
        update_status('No face in source path detected.', NAME)
        return False
    if not is_image(roop.globals.target_path) and not is_video(roop.globals.target_path):
        update_status('Select an image or video for target path.', NAME)
        return False
    return True
```

**Validation checks:**
1. Source must be an image
2. Source must contain at least one detectable face
3. Target must be an image or video

## Usage Examples

### Swap Single Face

```bash
python run.py -s source.jpg -t target.jpg -o output.jpg
```

### Swap Multiple Faces

```bash
python run.py -s source.jpg -t target.jpg -o output.jpg --many-faces
```

### Process Video

```bash
python run.py -s source.jpg -t video.mp4 -o output.mp4
```

## Thread Safety

The face swapper uses a threading lock to ensure thread-safe model initialization:

```python
THREAD_LOCK = threading.Lock()

with THREAD_LOCK:
    if FACE_SWAPPER is None:
        # Initialize model
```

This prevents race conditions when multiple threads try to load the model simultaneously.

## Resource Cleanup

After processing completes, the model is released to free memory:

```python
def post_process() -> None:
    global FACE_SWAPPER
    FACE_SWAPPER = None
```

<Note>
The model is lazy-loaded on first use and released after processing to optimize memory usage.
</Note>
