## Video Processing

### detect_fps()

Detects the frame rate of a video file.

```python
def detect_fps(target_path: str) -> float
```

**Parameters**

<ParamField path="target_path" type="string" required>
  Path to the video file
</ParamField>

**Returns**

<ResponseField name="fps" type="float">
  Frame rate of the video, or `30.0` if detection fails
</ResponseField>

**Description**

Uses `ffprobe` to extract the video's frame rate from the stream metadata.

**Example**

```python
from roop.utilities import detect_fps

fps = detect_fps('video.mp4')
print(f"Video FPS: {fps}")  # Video FPS: 29.97
```

### extract_frames()

Extracts all frames from a video to PNG images.

```python
def extract_frames(target_path: str) -> None
```

**Parameters**

<ParamField path="target_path" type="string" required>
  Path to the video file to extract frames from
</ParamField>

**Description**

Extracts frames to the temporary directory using FFmpeg:
- Output format: PNG with `rgb24` pixel format
- Naming pattern: `0001.png`, `0002.png`, etc.
- Output directory: `{video_dir}/temp/{video_name}/`

**Example**

```python
from roop.utilities import create_temp, extract_frames

target = 'video.mp4'
create_temp(target)
extract_frames(target)
print("Frames extracted")
```

### create_video()

Creates a video from extracted frames.

```python
def create_video(target_path: str, fps: float = 30.0) -> None
```

**Parameters**

<ParamField path="target_path" type="string" required>
  Original target video path (used to locate temp frames)
</ParamField>

<ParamField path="fps" type="float" default="30.0">
  Frame rate for the output video
</ParamField>

**Description**

Creates an MP4 video from PNG frames using FFmpeg:
- Codec: From `roop.globals.video_encoder`
- Quality: From `roop.globals.video_quality` (CRF)
- Pixel format: `yuv420p`
- Color space: BT.709 (converted from BT.601-6-625)

**Example**

```python
from roop.utilities import create_video, detect_fps
import roop.globals

roop.globals.video_encoder = 'libx264'
roop.globals.video_quality = 18

target = 'video.mp4'
fps = detect_fps(target)
create_video(target, fps)
print(f"Video created at {fps} fps")
```

### normalize_output_path()

Normalizes the output path based on source and target paths.

```python
def normalize_output_path(source_path: str, target_path: str, output_path: str) -> Any
```

**Parameters**

<ParamField path="source_path" type="string" required>
  Path to source image
</ParamField>

<ParamField path="target_path" type="string" required>
  Path to target image or video
</ParamField>

<ParamField path="output_path" type="string" required>
  User-specified output path (file or directory)
</ParamField>

**Returns**

<ResponseField name="normalized_path" type="string">
  Normalized output path
</ResponseField>

**Behavior**

If `output_path` is a directory, generates filename:
```
{output_dir}/{source_name}-{target_name}.{target_ext}
```

**Example**

```python
from roop.utilities import normalize_output_path

# Output to specific file
path = normalize_output_path('face.jpg', 'video.mp4', 'result.mp4')
print(path)  # result.mp4

# Output to directory
path = normalize_output_path('face.jpg', 'video.mp4', '/output/')
print(path)  # /output/face-video.mp4
```

## FFmpeg Utilities

### run_ffmpeg()

Executes FFmpeg with specified arguments.

```python
def run_ffmpeg(args: List[str]) -> bool
```

**Parameters**

<ParamField path="args" type="list[string]" required>
  FFmpeg command-line arguments (excluding 'ffmpeg')
</ParamField>

**Returns**

<ResponseField name="success" type="bool">
  `True` if FFmpeg command succeeded, `False` otherwise
</ResponseField>

**Description**

Runs FFmpeg with:
- Hardware acceleration enabled (`-hwaccel auto`)
- Log level from `roop.globals.log_level`
- Hidden banner

**Example**

```python
from roop.utilities import run_ffmpeg
import roop.globals

roop.globals.log_level = 'error'

success = run_ffmpeg([
    '-i', 'input.mp4',
    '-vf', 'scale=1280:720',
    '-y', 'output.mp4'
])

if success:
    print("Video processed")
```

## Download Utilities

### conditional_download()

Downloads files if they don't already exist locally.

```python
def conditional_download(download_directory_path: str, urls: List[str]) -> None
```

**Parameters**

<ParamField path="download_directory_path" type="string" required>
  Directory to download files into
</ParamField>

<ParamField path="urls" type="list[string]" required>
  List of URLs to download
</ParamField>

**Description**

Downloads each URL to the specified directory:
- Creates directory if it doesn't exist
- Skips files that already exist
- Shows progress bar with `tqdm`

**Example**

```python
from roop.utilities import conditional_download

models = [
    'https://example.com/model1.onnx',
    'https://example.com/model2.onnx'
]

conditional_download('./models', models)
print("Models downloaded")
```

## Complete Processing Example

```python
import roop.globals
from roop.utilities import (
    create_temp,
    extract_frames,
    get_temp_frame_paths,
    detect_fps,
    create_video,
    restore_audio,
    move_temp,
    clean_temp,
    is_video
)

# Configure
roop.globals.video_encoder = 'libx264'
roop.globals.video_quality = 18
roop.globals.keep_fps = True
roop.globals.keep_audio = True
roop.globals.keep_frames = False

target = 'input.mp4'
output = 'output.mp4'

# Process video
create_temp(target)
extract_frames(target)

frames = get_temp_frame_paths(target)
print(f"Processing {len(frames)} frames")

# ... process frames here ...

# Create output
fps = detect_fps(target) if roop.globals.keep_fps else 30.0
create_video(target, fps)

if roop.globals.keep_audio:
    restore_audio(target, output)
else:
    move_temp(target, output)

clean_temp(target)

if is_video(output):
    print("Processing complete!")
```
