---
title: Global Variables
description: Configuration state and runtime settings for Based Roop
---

The `roop.globals` module contains global configuration variables that control the behavior of the face swapping application. These variables are set during initialization and used throughout the processing pipeline.

## Path Variables

### source_path

Path to the source image containing the face to be swapped.

```python
source_path: str | None = None
```

**Usage**

```python
import roop.globals

roop.globals.source_path = '/path/to/source.jpg'
```

**Set By**: `parse_args()` from `--source` argument

---

### target_path

Path to the target image or video where the face will be swapped.

```python
target_path: str | None = None
```

**Usage**

```python
import roop.globals

roop.globals.target_path = '/path/to/target.mp4'
```

**Set By**: `parse_args()` from `--target` argument

---

### output_path

Path to the output file or directory where results will be saved.

```python
output_path: str | None = None
```

**Usage**

```python
import roop.globals

roop.globals.output_path = '/path/to/output.mp4'
```

**Set By**: `parse_args()` from `--output` argument (normalized via `normalize_output_path()`)

---

## Processing Configuration

### frame_processors

List of frame processor modules to apply during processing.

```python
frame_processors: List[str] = []
```

**Default**: `['face_swapper']`

**Available Processors**
- `face_swapper`: Swaps faces between source and target
- `face_enhancer`: Enhances face quality after swapping

**Usage**

```python
import roop.globals

roop.globals.frame_processors = ['face_swapper', 'face_enhancer']
```

**Set By**: `parse_args()` from `--frame-processor` argument

---

### many_faces

Whether to process all faces in a frame or just the first detected face.

```python
many_faces: bool | None = None
```

**Default**: `False`

**Behavior**
- `False`: Only processes the leftmost face (detected by `get_one_face()`)
- `True`: Processes all detected faces (detected by `get_many_faces()`)

**Usage**

```python
import roop.globals

roop.globals.many_faces = True  # Process all faces
```

**Set By**: `parse_args()` from `--many-faces` flag

---

## Video Settings

### keep_fps

Whether to preserve the original video's frame rate.

```python
keep_fps: bool | None = None
```

**Default**: `False`

**Behavior**
- `True`: Detects and preserves original FPS from target video
- `False`: Uses default 30.0 FPS for output

**Usage**

```python
import roop.globals

roop.globals.keep_fps = True
```

**Set By**: `parse_args()` from `--keep-fps` flag

---

### keep_audio

Whether to preserve the original audio from the target video.

```python
keep_audio: bool | None = None
```

**Default**: `True`

**Usage**

```python
import roop.globals

roop.globals.keep_audio = True
```

**Set By**: `parse_args()` from `--keep-audio` flag

---

### keep_frames

Whether to keep temporary frame files after processing.

```python
keep_frames: bool | None = None
```

**Default**: `False`

**Behavior**
- `False`: Temporary frames are deleted after video creation
- `True`: Temporary frames are preserved in the temp directory

**Usage**

```python
import roop.globals

roop.globals.keep_frames = True  # Keep temp frames for debugging
```

**Set By**: `parse_args()` from `--keep-frames` flag

---

### video_encoder

Video codec to use for encoding output videos.

```python
video_encoder: str | None = None
```

**Default**: `'libx264'`

**Available Encoders**
- `libx264`: H.264 codec (widely compatible)
- `libx265`: H.265/HEVC codec (better compression)
- `libvpx-vp9`: VP9 codec (open format)

**Usage**

```python
import roop.globals

roop.globals.video_encoder = 'libx265'  # Use H.265
```

**Set By**: `parse_args()` from `--video-encoder` argument

---

### video_quality

Constant Rate Factor (CRF) value for video quality.

```python
video_quality: int | None = None
```

**Default**: `18`

**Range**: 0-51
- Lower values = better quality, larger file size
- Higher values = lower quality, smaller file size
- Recommended: 18-28

**Usage**

```python
import roop.globals

roop.globals.video_quality = 23  # Good balance
```

**Set By**: `parse_args()` from `--video-quality` argument

---

## System Resources

### max_memory

Maximum amount of RAM to use (in GB).

```python
max_memory: int | None = None
```

**Default**: 
- macOS: `4` GB
- Other platforms: `16` GB

**Usage**

```python
import roop.globals

roop.globals.max_memory = 8  # Use 8 GB max
```

**Set By**: `parse_args()` from `--max-memory` argument

---

### execution_providers

ONNX Runtime execution providers to use for inference.

```python
execution_providers: List[str] = []
```

**Default**: `['CPUExecutionProvider']`

**Common Providers**
- `CPUExecutionProvider`: CPU inference
- `CUDAExecutionProvider`: NVIDIA GPU inference
- `ROCMExecutionProvider`: AMD GPU inference
- `DmlExecutionProvider`: DirectML (Windows)
- `TensorrtExecutionProvider`: TensorRT optimization

**Usage**

```python
import roop.globals

roop.globals.execution_providers = ['CUDAExecutionProvider', 'CPUExecutionProvider']
```

**Set By**: `parse_args()` from `--execution-provider` argument (decoded via `decode_execution_providers()`)

---

### execution_threads

Number of threads to use for inference.

```python
execution_threads: int | None = None
```

**Default**: 
- DML/ROCM providers: `1`
- Other providers: `8`

**Usage**

```python
import roop.globals

roop.globals.execution_threads = 4
```

**Set By**: `parse_args()` from `--execution-threads` argument

---

## UI and Logging

### headless

Whether the application is running in headless mode (without GUI).

```python
headless: bool | None = None
```

**Default**: `None`

**Behavior**
- `True`: Runs without GUI, directly processes files
- `False`: Launches GUI window for interactive use

**Auto-Detection**: Set to `True` if any of `source_path`, `target_path`, or `output_path` are provided

**Usage**

```python
import roop.globals

if roop.globals.headless:
    print("Running in headless mode")
```

**Set By**: `parse_args()` based on presence of path arguments

---

### log_level

FFmpeg log verbosity level.

```python
log_level: str = 'error'
```

**Default**: `'error'`

**Available Levels**
- `quiet`: No output
- `error`: Errors only
- `warning`: Warnings and errors
- `info`: Informational messages
- `verbose`: Detailed output
- `debug`: Debug information

**Usage**

```python
import roop.globals

roop.globals.log_level = 'info'  # More verbose output
```

## Complete Example

```python
import roop.globals
import roop.core

# Configure processing
roop.globals.source_path = 'face.jpg'
roop.globals.target_path = 'video.mp4'
roop.globals.output_path = 'output.mp4'
roop.globals.frame_processors = ['face_swapper', 'face_enhancer']
roop.globals.many_faces = True
roop.globals.keep_fps = True
roop.globals.keep_audio = True
roop.globals.video_encoder = 'libx264'
roop.globals.video_quality = 18
roop.globals.execution_providers = ['CUDAExecutionProvider']
roop.globals.execution_threads = 8
roop.globals.max_memory = 16
roop.globals.headless = True

# Start processing
if roop.core.pre_check():
    roop.core.limit_resources()
    roop.core.start()
```
