## Overview

The face swapper processor (`roop.processors.frame.face_swapper`) is the core component for swapping faces in images and videos. It uses the InsightFace inswapper_128 ONNX model to perform high-quality face swapping.

**Module:** `roop.processors.frame.face_swapper`

**Model:** `inswapper_128.onnx`

## Core Functions

### get_face_swapper()

Retrieves or initializes the face swapper model instance. Uses thread-safe singleton pattern to ensure only one model instance is loaded.

```python
def get_face_swapper() -> Any
```

**Returns:** InsightFace model instance configured with the execution providers from `roop.globals.execution_providers`

**Thread Safety:** Yes - uses `THREAD_LOCK` to prevent race conditions

**Example:**

```python
from roop.processors.frame import face_swapper

model = face_swapper.get_face_swapper()
# Model is cached and reused across calls
```

### swap_face()

Performs the actual face swap operation between a source face and target face in a frame.

```python
def swap_face(source_face: Face, target_face: Face, temp_frame: Frame) -> Frame
```

<ParamField path="source_face" type="Face" required>
  The source face to transplant onto the target
</ParamField>

<ParamField path="target_face" type="Face" required>
  The target face to be replaced in the frame
</ParamField>

<ParamField path="temp_frame" type="Frame" required>
  The frame (numpy array) containing the target face
</ParamField>

**Returns:** Modified frame with the face swapped

**Example:**

```python
import cv2
from roop.face_analyser import get_one_face
from roop.processors.frame import face_swapper

source_frame = cv2.imread('source.jpg')
target_frame = cv2.imread('target.jpg')

source_face = get_one_face(source_frame)
target_face = get_one_face(target_frame)

result = face_swapper.swap_face(source_face, target_face, target_frame)
cv2.imwrite('output.jpg', result)
```

### process_frame()

Processes a single frame, swapping faces based on the `many_faces` global setting.

```python
def process_frame(source_face: Face, temp_frame: Frame) -> Frame
```

<ParamField path="source_face" type="Face" required>
  The source face to use for swapping
</ParamField>

<ParamField path="temp_frame" type="Frame" required>
  The frame to process
</ParamField>

**Returns:** Processed frame with face(s) swapped

**Behavior:**
- If `roop.globals.many_faces` is `True`, swaps all detected faces in the frame
- If `roop.globals.many_faces` is `False`, swaps only the first detected face
- Returns original frame unchanged if no faces are detected

**Example:**

```python
import cv2
import roop.globals
from roop.face_analyser import get_one_face
from roop.processors.frame import face_swapper

# Swap single face
roop.globals.many_faces = False
source_face = get_one_face(cv2.imread('source.jpg'))
frame = cv2.imread('target.jpg')

result = face_swapper.process_frame(source_face, frame)

# Swap all faces
roop.globals.many_faces = True
result_multi = face_swapper.process_frame(source_face, frame)
```

### process_frames()

Processes multiple frames in batch, typically used as a callback for video processing.

```python
def process_frames(source_path: str, temp_frame_paths: List[str], update: Callable[[], None]) -> None
```

<ParamField path="source_path" type="str" required>
  Path to the source image containing the face to swap
</ParamField>

<ParamField path="temp_frame_paths" type="List[str]" required>
  List of paths to temporary frame files to process
</ParamField>

<ParamField path="update" type="Callable[[], None]" required>
  Callback function to invoke after processing each frame (for progress tracking)
</ParamField>

**Returns:** None - modifies frames in-place by overwriting the temporary files

**Example:**

```python
from roop.processors.frame import face_swapper

def progress_callback():
    print("Frame processed")

frame_paths = ['temp/frame_001.jpg', 'temp/frame_002.jpg', 'temp/frame_003.jpg']
face_swapper.process_frames('source.jpg', frame_paths, progress_callback)
```

### process_image()

Processes a single image file, performing face swap from source to target.

```python
def process_image(source_path: str, target_path: str, output_path: str) -> None
```

<ParamField path="source_path" type="str" required>
  Path to the source image containing the face to swap
</ParamField>

<ParamField path="target_path" type="str" required>
  Path to the target image where the face will be swapped
</ParamField>

<ParamField path="output_path" type="str" required>
  Path where the result image will be saved
</ParamField>

**Returns:** None - writes the result to the output path

**Example:**

```python
from roop.processors.frame import face_swapper

face_swapper.process_image(
    source_path='me.jpg',
    target_path='celebrity.jpg',
    output_path='result.jpg'
)
```

### process_video()

Processes a video by swapping faces in all frames.

```python
def process_video(source_path: str, temp_frame_paths: List[str]) -> None
```

<ParamField path="source_path" type="str" required>
  Path to the source image containing the face to swap
</ParamField>

<ParamField path="temp_frame_paths" type="List[str]" required>
  List of paths to temporary frame files extracted from the video
</ParamField>

**Returns:** None - delegates to `roop.processors.frame.core.process_video()` with multi-threading support

**Example:**

```python
from roop.processors.frame import face_swapper

# Assume frames are already extracted
frame_paths = ['temp/frame_001.jpg', 'temp/frame_002.jpg', ...]
face_swapper.process_video('source.jpg', frame_paths)
```

## Lifecycle Functions

### pre_check()

Validates prerequisites and downloads required model files.

```python
def pre_check() -> bool
```

**Returns:** `True` if all prerequisites are met

**Behavior:**
- Downloads `inswapper_128.onnx` model from HuggingFace if not present
- Model URL: `https://huggingface.co/henryruhs/roop/resolve/main/inswapper_128.onnx`
- Downloads to: `../models/` directory relative to the module

**Example:**

```python
from roop.processors.frame import face_swapper

if face_swapper.pre_check():
    print("Ready to swap faces")
```

### pre_start()

Validates that source and target paths are valid and contain faces.

```python
def pre_start() -> bool
```

**Returns:** `True` if validation passes, `False` otherwise

**Validation Rules:**
1. Source path must be an image file
2. Source image must contain at least one detectable face
3. Target path must be either an image or video file

**Example:**

```python
import roop.globals
from roop.processors.frame import face_swapper

roop.globals.source_path = 'source.jpg'
roop.globals.target_path = 'target.mp4'

if face_swapper.pre_start():
    # Proceed with processing
    pass
else:
    # Handle validation errors
    print("Validation failed - check status messages")
```

### post_process()

Cleans up resources by releasing the face swapper model from memory.

```python
def post_process() -> None
```

**Returns:** None

**Behavior:**
- Sets the global `FACE_SWAPPER` instance to `None`
- Allows garbage collection of the model to free memory

**Example:**

```python
from roop.processors.frame import face_swapper

# After processing is complete
face_swapper.post_process()
```

## Integration Example

Complete workflow for face swapping in an image:

```python
import roop.globals
from roop.processors.frame import face_swapper

# Set global configuration
roop.globals.source_path = 'source.jpg'
roop.globals.target_path = 'target.jpg'
roop.globals.output_path = 'result.jpg'
roop.globals.many_faces = False
roop.globals.execution_providers = ['CPUExecutionProvider']

# Lifecycle: pre-check
if not face_swapper.pre_check():
    print("Model download failed")
    exit(1)

# Lifecycle: pre-start validation
if not face_swapper.pre_start():
    print("Validation failed")
    exit(1)

# Process the image
face_swapper.process_image(
    roop.globals.source_path,
    roop.globals.target_path,
    roop.globals.output_path
)

# Lifecycle: cleanup
face_swapper.post_process()

print(f"Face swap complete: {roop.globals.output_path}")
```

## Type Definitions

The processor uses type aliases from `roop.typing`:

- **Face:** InsightFace face detection result object
- **Frame:** Numpy array representing an image (typically from `cv2.imread()`)

## Dependencies

- `insightface` - Face detection and swapping model
- `cv2` (OpenCV) - Image I/O and manipulation
- `roop.face_analyser` - Face detection utilities (`get_one_face`, `get_many_faces`)
- `roop.globals` - Global configuration and state
- `roop.processors.frame.core` - Core video processing utilities
